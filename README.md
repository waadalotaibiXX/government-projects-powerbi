# government-projects-powerbi
End-to-end data analysis of government projects using Python for data preprocessing and Power BI for interactive dashboards and insights.
 # Government Projects Analysis (Saudi Arabia)

## Overview
This project analyzes ongoing government projects in Saudi Arabia using Python for data preprocessing and Power BI for data visualization.

## Data Source
- Saudi Water Authority – Open Data Portal  
- Dataset: Projects under implementation for the central sector

## Tools & Technologies
- Python (Pandas, Jupyter Notebook)
- Power BI
- CSV / Excel datasets

## Project Workflow
1. Raw data collection from the open data portal
2. Data cleaning and preprocessing using Python
3. Exporting cleaned data to CSV
4. Building interactive dashboards using Power BI

## Files
- `raw_projects_data.csv` → Raw dataset
- `cleaned_projects_data.csv` → Cleaned dataset
- `saudi_projects_analysis.ipynb` → Data preprocessing & analysis
- `Government Projects Dashboard.pbix` → Power BI dashboard

## Key Insights
- Asset type distribution of projects
- Yearly trend of government projects
- Dominant asset categories
